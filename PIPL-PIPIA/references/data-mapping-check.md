# 数据映射字段完整性检查

本文件定义GB/T 39335-2020附录C要求的数据映射字段完整性检查标准。

---

## ⛔ 数据映射字段完整性规则

### 强制规则

```
┌─────────────────────────────────────────────────────────────────────┐
│  ⛔ 数据映射字段完整性规则                                              │
├─────────────────────────────────────────────────────────────────────┤
│  规则1：每组处理活动必须创建独立数据映射表                               │
│  规则2：所有数据映射表必须包含完整字段                                   │
│  规则3：字段格式必须在所有处理活动间保持一致                             │
│                                                                      │
│  ⛔ 禁止行为：                                                         │
│  □ 处理活动A数据映射表字段完整，处理活动B字段缺失                        │
│  □ 跳过"存储方式/加密措施"等关键字段                                    │
│  □ 不同处理活动的数据映射表格式不统一                                   │
└─────────────────────────────────────────────────────────────────────┘
```

---

## GB/T 39335-2020附录C数据映射字段清单

### 完整字段列表

每组处理活动的数据映射表**必须**包含以下字段：

| # | 字段名称 | 字段说明 | 是否必填 | 示例 |
|---|---------|---------|---------|------|
| 1 | 个人信息类型 | 具体的个人信息字段名称 | 必填 | 手机号、身份证号、位置信息 |
| 2 | 是否敏感个人信息 | 是否属于PIPL §28定义的敏感PI（识别依据：PIPL §28法定定义 + 《网络安全标准实践指南》第3-4章识别规则/类别 + 附录A示例） | 必填 | 是/否 |
| 3 | 处理目的 | 收集和处理该信息的目的 | 必填 | 账户注册、订单配送、个性化推荐 |
| 4 | 法律依据 | 处理该信息的合法性基础 | 必填 | §13(一)同意、§13(五)履行合同 |
| 5 | 收集来源 | 信息的收集渠道 | 必填 | 用户主动填写、第三方接入 |
| 6 | 收集方式 | 信息的收集方法 | 必填 | 表单提交、API接口、Cookie |
| 7 | 控制者 | 个人信息处理者名称 | 必填 | XX科技有限公司 |
| 8 | 处理者 | 实际处理个人信息的主体 | 如适用 | XX云计算服务有限公司 |
| 9 | 是否跨境 | 是否向境外提供 | 必填 | 是/否 |
| 10 | 境外接收方 | 如跨境，接收方名称 | 如跨境必填 | XX公司（美国） |
| 11 | 存储位置 | 信息的存储地点 | 必填 | XX数据中心（上海） |
| 12 | 存储方式/加密措施 | 存储的安全措施 | 必填 | 数据库加密（AES-256） |
| 13 | 传输方式/加密措施 | 传输的安全措施 | 必填 | HTTPS（TLS 1.3） |
| 14 | 存储期限 | 信息的保留期限 | 必填 | 用户注销后30天删除 |
| 15 | 删除/匿名化方式 | 信息的处置方式 | 必填 | 物理删除、匿名化处理 |
| 16 | 访问权限 | 有权访问该信息的角色 | 必填 | 客服人员（只读） |
| 17 | 第三方共享情况 | 是否与第三方共享 | 必填 | 是/否，如是的共享方名称 |
| 18 | 安全措施 | 已采取的保护措施 | 必填 | 访问控制、日志审计、脱敏 |

## 常见缺失字段示例

| 缺失字段 | 影响 | 建议补充内容 |
|---------|------|-------------|
| 存储方式/加密措施 | 无法评估存储安全风险 | 补充加密标准（如AES-256、数据库加密） |
| 传输方式/加密措施 | 无法评估传输安全风险 | 补充传输协议（如HTTPS/TLS 1.3） |
| 存储期限 | 无法评估合规性 | 补充保留期限和删除时机 |
| 删除/匿名化方式 | 无法评估处置合规性 | 补充删除方法或匿名化技术 |
